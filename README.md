### Pre-requisites
- Install docker

### How to run
## Go Playground Service
```
make docker-run
```

## Go Playground Webapp
``` 
cd webapp;
make docker-run
```

### Debugging
## Build and Push Go Playground Service
``` 
make docker-build;
make docker-push
```

## Build and Push go Playground Webapp
``` 
cd webapp;
make docker-build;
make docker-push
```
